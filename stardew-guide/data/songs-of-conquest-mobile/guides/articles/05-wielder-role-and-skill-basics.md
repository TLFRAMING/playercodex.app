# Songs of Conquest Mobile Wielder Guide: Main Wielder, Support Wielder, and Skill Direction

A Wielder is not just the unit that moves your army around the map. In Songs of Conquest Mobile, Wielder decisions shape exploration tempo, army strength, spell access, and how safely you can take fights.

The cleanest beginner approach is to give each Wielder a job before you start spending attention on every possible upgrade.

## Start With One Main Wielder

Your main Wielder should carry the army that handles the most important fights. This is the Wielder you protect, develop, and use to open the map.

A good main Wielder plan answers:

- Which army is this Wielder supporting?
- What kind of fights should this Wielder take?
- Which resources or towns does this Wielder need to reach next?
- What kind of spell support would make those fights safer?

If you cannot answer those questions, do not rush into difficult battles. The Wielder may be moving, but the run is not yet organized.

## Use Support Wielders For Map Work

Support Wielders can help the map feel less cramped, but they should not steal focus from the main plan.

Good support tasks include:

- collecting safer resources
- moving toward low-risk pickups
- helping town development stay connected to the main army
- scouting a nearby direction before the main army commits

Poor support use usually looks like this:

- sending a weak Wielder into a fight that matters
- splitting army strength before the main Wielder is stable
- chasing distant rewards while the main objective waits
- creating extra decisions without solving the current bottleneck

Support Wielders are useful when they reduce pressure. They are a problem when they create a second campaign plan that competes with the first.

## Match Skills To The Army's Job

Skill direction should support what the Wielder is actually doing. A Wielder carrying the main army needs different priorities from a Wielder collecting safe resources or preparing future options.

Use this decision order:

1. Decide the Wielder's job.
2. Check what the army is struggling with.
3. Choose development that helps that job happen more safely.
4. Ignore upgrades that look interesting but do not help the current role.

This prevents scattered Wielder growth. A skill that is useful in theory can still be wrong if it does not help the army, map route, or battle plan you are actually using.

## Connect Wielders To Essence

Wielders and Essence should be planned together. If you think about them separately, fights become harder to read.

Before a major fight, check:

- Does this Wielder have a clear spell purpose?
- Does the army help create the Essence pattern you want?
- Are you relying on spells to fix a fight that should have been avoided?
- Would a different Wielder be better used for map work while the main army prepares?

If spell availability keeps surprising you, review the [Essence guide](/songs-of-conquest-mobile/guides/essence-spellcasting-basics) and start checking the Wielder's role before each serious battle.

## Avoid The Disposable Wielder Mistake

New players sometimes treat Wielders as expendable scouts. That can create a hidden cost: lost tempo, awkward army movement, and fights that must be delayed because the right Wielder is out of position.

Before sending a Wielder away from the main plan, ask:

1. What will this Wielder accomplish in the next few turns?
2. Is the path safe enough for the army strength attached to them?
3. Will this movement help the main objective, or only reveal more distractions?

If the answer is unclear, keep the Wielder closer to useful map pressure.

## Common Beginner Mistakes

- Developing a Wielder around random upgrades instead of a clear role.
- Splitting army strength too early.
- Using support Wielders to start fights they cannot finish cleanly.
- Treating the main Wielder's army and spell plan as separate systems.
- Exploring wide areas before the current objective is stable.

## A Simple Wielder Routine

At the start of each turn, give every active Wielder one sentence:

- "This Wielder is opening the next objective."
- "This Wielder is collecting safe resources."
- "This Wielder is returning toward the army."
- "This Wielder is waiting because the army is not ready."

If you cannot write that sentence, the Wielder probably needs a clearer job. A clear role makes campaign turns faster, battles safer, and mobile sessions easier to resume.
